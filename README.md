# Gundou-Project

## О проекте
**Gundou** — это веб-интерфейс для управления Discord-ботом. Панель предоставляет администратору контроль над ботом, просмотр статистики и управление настройками без необходимости прямого доступа к серверу или консоли.

## Возможности

### **Управление сообщениями**
- Настройка приветственных сообщений
- Отправка очереди сообщений и эмбедов в текстовые каналы

###  **Управление участниками**
- Просмотр списка участников
- Управление ролями 
- Модерация канала

###  **Статус-бар**
- Логирование последних действий, произведенных с ботом

##  Запуск

```bash
# 1. Убедитесь что файл app/config.json существует (получить у администратора репоз-я)

# 2. Убедитесь в корректности guildID, наличии прав для бота и корректного token в config.json

# 3. Запустите start.bat

# 4. Перейдите в бразер по адресу:
#    http://localhost:5665/main?token=1234 (1234 - фиктивный токен)
```
